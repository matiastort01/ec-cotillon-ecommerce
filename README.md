## 📌 Sobre el Proyecto

Este proyecto consiste en una plataforma de E-commerce diseñada para gestionar un catálogo de productos de forma eficiente. Implementa una arquitectura robusta donde se prioriza la experiencia del usuario y la integridad de los datos en los procesos de compra y administración.

El sistema utiliza el motor de plantillas Blade para el renderizado de vistas y Eloquent ORM para una gestión ágil y segura de la base de datos relacional.

## 🏗️ Funcionalidades Principales

* **Gestión de Catálogo:** Visualización dinámica de productos con soporte para categorías, imágenes y descripciones detalladas.
* **Sistema de Autenticación y Roles:** Implementación de seguridad para diferenciar entre clientes (visualización y compra) y administradores (gestión de stock y órdenes).
* **Filtros Dinámicos:** Buscador y filtrado de productos por precio, categoría y disponibilidad para mejorar la navegación.
* **Panel de Administración (Back-office):** Interfaz exclusiva para la gestión CRUD (Crear, Leer, Actualizar, Borrar) de productos y monitoreo de actividad.

## ⚙️ Características Técnicas

* **Arquitectura MVC:** Separación clara de responsabilidades siguiendo los estándares de Laravel para facilitar la escalabilidad.
* **Eloquent ORM:** Uso de modelos para interactuar con la base de datos de forma segura, evitando inyecciones SQL y optimizando consultas.
* **Motor de Plantillas Blade:** Implementación de vistas dinámicas mediante herencia de plantillas, directivas de control y componentes reutilizables.
* **Persistencia con phpMyAdmin:** Gestión de base de datos relacional MySQL optimizada y administrada localmente.
* **Entorno de Desarrollo:** Configurado y testeado íntegramente utilizando el stack **XAMPP**.
* **Middleware de Seguridad:** Control de acceso a rutas protegidas según el perfil de usuario autenticado.
* **Diseño Adaptable:** Interfaz responsive construida con Bootstrap para asegurar la funcionalidad en dispositivos móviles y desktop.
